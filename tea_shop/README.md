# TeaShop Database

Relational database for a tea shop chain, designed to practice SQL skills.

## Highlights
- Tables with **PKs** and **FKs**
- **One-to-many** (Shop → Manager) and **many-to-many** (Tea ↔ Shop)
- **Stored procedures** for safe inserts
- **View** combining Tea, Shop, and Manager
- Sample data included
